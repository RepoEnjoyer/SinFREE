SinFree is a free, open-source DLL extension for Counter-Strike 2 that provides visual enhancements and performance monitoring for offline practice and private server use. This is the free version of the full Sin Menu designed for players who want to explore basic features with heavily minimalized functionality and settings before committing to the complete package which is heavily improved.

Default key is DEL to open the Menu
End Terminates the Menu

Join the discord for Extra Support and More Resources! - https://discord.gg/ve2YhZphnw

## Features
**Free version includes:**

- Entity information display (Box, Name, Tracer Lines)
- ImGui configuration menu (press F1 to open)
- Aimbot (Reduced Customizations and no Configs) - Premium has much more customization
- Configurable input smoothing for testing mechanical skill
- Lightweight design with minimal performance impact
- Undetected

**Requirements:**
- Windows 10/11 (x64)
- Visual C++ Redistributable 2022 - Install Here https://learn.microsoft.com/en-us/answers/questions/4130948/looking-for-microsoft-visual-c-2022-redistributabl
- CS2 installed (Steam version)

⚠️ **Important**: This tool is intended for educational purposes only. Use only in offline/practice environments or private servers. Not for official online matches.

## Building from Source

**To compile the DLL yourself incase you want the base or want to read the code**

git clone https://github.com/RepoEnjoyer/sinfree.git
cd sinfree

Open SinFree.sln in Visual Studio 2022 and build the Release | x64 configuration. The output DLL will be in x64\Release\.

**Dependencies For SinFREE:**
- MinHook – function hooking
- ImGui – user interface
- Output - (CS2's Latest offset dump needed if game updates, Developer will upkeep here)
  
Run git submodule update --init --recursive to fetch dependencies.
